Réservation de Vélib à Paris

Comment l'utiliser ?

Etape 1

Des marqueurs indiquent le nombre de stations présentes dans un périmètre donné. Zoomer ou cliquer sur un marqueur pour voir les stations

Etape 2

Cliquer sur une station pour en avoir les détails dans le panneau d'informations (adresse de la station, nombre de vélos disponibles)

Etape 3

Dans le panneau "Détails de la station" à droite de la carte, cliquer sur "Réserver" afin d'ouvrir le panneau de réservation

Etape 4

Signer dans le panneau prévu à cet effet, cliquer sur "Effacer" pour recommencer si besoin, puis cliquer sur "Valider"

Etape 5

Votre réservation est valable pendant 20min avant confirmation, cliquer sur "Annuler" pour annuler votre réservation. Si vous décidez de réserver dans une autre station, cela annule automatiquement votre réservation précédente
